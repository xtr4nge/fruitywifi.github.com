New Release v2.0
Overview
FruityWifi es una herramienta de codigo abierto para auditoria de redes wireless. Permite desplegar rapida y facilmente ataques avanzados utilizando la interfaz web o enviandole mensajes.

Inicialmente la aplicacion fue creada para Raspberry-Pi, pero puede ser instalada en cualquier sistema basado en Debian.

FruityWifi v2.0 trae muchas novedades. Una nueva interfaz, nuevos modulos, soporte para chipsets Realtek, soporte para Mobile Broadband (3G/4G), un nuevo panel de configuracion, y mas.
Un panel de configuracion mas flexible. Ahora es posible utilizar FruityWifi combinando diferentes redes:

- Ethernet Ethernet,
- Ethernet 3G/4G,
- Ethernet Wifi,
- Wifi Wifi,
- Wifi 3G/4G, etc.

Entre las nuevas opciones de este panel, es posible cambiar el AP de Hostapd a Airmon-ng pudiendo utilizar mas chipsets como por ejemplo placas Realtek.

Cada una de las interfaces es customizable, pudiendo dejar la configuracion actual o cambiandola completamente.
Nuevos Modulos
FruityWifi se compone de modulos que posibilitan su escalabilidad. Estos modulos pueden ser instalados para agregar nuevas funcionalidades.

Entre los modulos disponibles se encuentran URLsnarf, DNSspoof, Kismet, mdk3, ngrep, nmap, Squid3 y SSLstrip (con funcionalidades para inyectar codigo), Captive Portal, AutoSSH, Meterpreter, Tcpdump y mas.
AutoSSH nos permite iniciar una conexion ssh reversa, reanudandola en caso de que detecte que se corte o se pare el trafico. Es especialmente util para mantener una conexion permantente con la FruityWifi.

Meterpreter es una impresionante utilidad que permite obtener una gran cantidad de informacion sobre un objetivo comprometido, asi como tambien manipular procesos del sistema y/o terminarlos. Este modulo nos permite desde la FruityWifi, comprometer nuevos equipos y utilizarlos como punto de entrada a otros dispositivos y redes.

Nessus es un programa de escaneo de vulnerabilidades. Con este modulo podemos escanear equipos desde la FruityWifi, sin necesidad de usar el panel de Nessus. Podemos conocer las vulnerabilidades de cada uno de los equipos conectados a nuestro dispositivo para tener mas detalles de nuestra superficie o comprometerlos.
Entre las nuevas caracteristicas, podemos encontrar que FrutyWifi ahora soporta Mobile Broadband (3G/4G). Podemos utilizar este modulo para conectar nuestro dongle 3G/4G y proveer a la FruityWifi the accesso a internet sin necesidad de conectarla a una red Wifi o Ethernet.
Tcpdump es un herramienta cuya utilidad principal es analizar el trafico que circula por la red. Con este modulo podemos interceptar el trafico que pasa por nuestro dispositivo, filtrarlo y/o almacenarlo para un analisis posterior.
Ettercap es una utilidad que nos permite capturar el trafico que circula por la red. Utilizado este modulo podemos realizar attaques MITM por medio de ARP poisoning.
